# PrimeKit

A production-grade **Next.js SaaS boilerplate**. Auth, billing, organizations, seats, and RBAC wired up and tested, so the boring infrastructure stops eating launch timelines.

**[primekit.dev](https://primekit.dev)** · [Docs](https://primekit.dev/docs) · [Blog](https://primekit.dev/blog)

- Stripe billing and subscriptions, ready to go
- Type-safe API (oRPC), Drizzle ORM, and a strict TypeScript setup
- shadcn/ui components on Tailwind, dark mode included
